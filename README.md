## 1 задача

Напишите и экспортируйте функцию `calculatePower()`, которая возвращает значение силы тока.

**Условия**

- Для получения результата необходимо напряжение разделить на сопротивление.

**Параметры**

- *a*: напряжение.
- *b*: сопротивление.

**Пример использования:**

```javascript
calculatePower(120, 12); // 10
calculatePower(100, 50); // 2
calculatePower(5, 2.5); // 2 
```

## 2 задача

Напишите и экспортируйте функцию `swapCase()`, которая возвращает строку с инвертируемым регистром.

**Параметры**

- *text*: строка, в которой необходимо инвертировать регистр букв.

**Пример использования:**

```javascript
swapCase('John'); // jOHN
swapCase('IS'); // is
swapCase('aDmIn');// AdMiN
```

## 3 задача

Напишите и экспортируйте функцию `formatAddress()`, которая возвращает отформатированный адрес.

**Условия**

- Функция должна вернуть строку следующего формата: `'Улица, Город, Почтовый индекс, Страна'`.

**Параметры**

- *street*: улица.
- *city*: город.
- *country*: страна.
- *index*: почтовый индекс.

**Пример использования:**

```javascript
formatAddress('Main Street', 'Springfield', 'USA', '12345'); // Main Street, Springfield, 12345, USA
formatAddress('Невский проспект', 'Санкт-Петербург', 'Россия', '190000'); // Невский проспект, Санкт-Петербург, 190000, Россия
formatAddress('Проспект Гагарина', 'Нижний Новгород', 'Россия', '607800'); // Проспект Гагарина, Нижний Новгород, 190000, Россия
```

## 4 задача

Напишите и экспортируйте функцию `getTimeOfDay()`, которая возвращает строку, описывающую текущее время суток.

**Условия**

- Если час от 0 до 5 включительно - возвращает "Ночь".
- Если час от 6 до 11 включительно - возвращает "Утро".
- Если час от 12 до 17 включительно - возвращает "День".
- Если час от 18 до 23 включительно - возвращает "Вечер".
- Если входное значение не является числом или не находится в указанном диапазоне, функция должна вернуть 0.

**Параметры**

- *time*: время в часах.

**Пример использования:**

```javascript
getTimeOfDay(2); // Ночь
getTimeOfDay(7); // Утро
getTimeOfDay(24);  // 0
```

## 5 задача

Напишите и экспортируйте функцию `countLetter()`, которая возвращает количество повторений символа в строке.

**Параметры**

- *text*: строка, в которой считаются символ.
- *symb*: символ, количество которого надо вернуть.

**Пример использования:**

```javascript
countLetter('word', 'o'); // 1
countLetter('hello', 'l'); // 2
countLetter('bounce', 'a'); // 0
```
